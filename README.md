# Lightning (procedural animation) - Python

Génère un effet "lightning" en frames PNG puis assemble une vidéo MP4 via FFmpeg.

## Dépendances
- Python 3
- numpy
- matplotlib
- ffmpeg

## Installation
pip install numpy matplotlib

## Usage
python3 lightning.py

## Output
- frames/frame_0000.png ...
- lightning.mp4
